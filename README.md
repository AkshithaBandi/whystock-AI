# WhyStock AI 📈🤖

AI-powered stock analysis platform that explains stock price movements using real-time data and local LLM (Ollama).

## 🚀 Features
- 📊 Real-time stock data
- 🧠 AI-generated explanations
- 📰 Latest news integration
- ⚡ Local AI (no API cost)
- ⭐ Watchlist (coming soon)

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: FastAPI
- AI: Ollama (Mistral/LLaMA3)
- Data: yfinance + News API

## 💡 Highlights
- Built full-stack architecture
- Integrated local LLM (no API dependency)
- Implemented caching for optimization
